🌸 MangaWorld Catalog

MangaWorld Catalog is an interactive web application created for manga enthusiasts who want to explore and discover popular manga series. Each entry provides complete details, including the author, genre, number of volumes, publisher, price, stock availability, reading direction, page count, and a summary. Users can hover over the manga covers to instantly reveal all the information, making it easy to choose their next favorite story. The project also includes XML data files for structured access to each manga’s details.

🔹 Live Preview

Hovering over each manga cover reveals detailed information, and users can explore XML data for deeper insights. The catalog currently features titles such as Jujutsu Kaisen by Gege Akutami and Solo Leveling by Chugong, with summaries that allow readers to quickly grasp the storyline and themes before diving in.

✨ Features

MangaWorld Catalog combines an interactive card layout with responsive design, providing a clean and user-friendly interface. The navigation bar allows quick access to XML files for each manga, while the footer provides additional navigation and information about the project. Each manga card is designed to be engaging and informative, offering readers all the details they need in one glance.

🛠 Technologies

The project is built using HTML5 for semantic structure and content organization, CSS3 for styling and responsive layout, and XML files for structured data. This combination ensures that the catalog is visually appealing, easy to navigate, and fully interactive.

📂 Usage

To use the catalog, simply open index.html in a web browser. Browsing through the manga collection is intuitive, as users can hover over the cards to view detailed information and summaries. XML files are directly accessible through the navigation bar or footer, providing structured data for each manga.

📖 Manga Included

The catalog currently features Jujutsu Kaisen, where Yuji Itadori becomes the host of a powerful curse to protect his friends, learning to master his abilities while facing dangerous curses and sorcerers. It also includes Solo Leveling, following Sung Jin-Woo, the weakest hunter, who survives a deadly dungeon and gains a system that allows him to level up infinitely, facing increasingly powerful foes and uncovering secrets that could change the fate of the world.

📝 Project Information

MangaWorld Catalog is a web development project completed as part of a DAW course in 2026. Its purpose is to provide an interactive platform for discovering and exploring manga, combining visual appeal with structured data and an intuitive interface.
